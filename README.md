**Live URL** — https://kp-devfolio.vercel.app/

This isn't your typical recruiter-friendly portfolio.
This is an experimental portfolio where creativity over convention.

I've always found most portfolios repetitive and template-driven, so instead of following that path, I built this to push my own limits as a frontend developer.

**Tech Stack**
- Frontend: React
- Styling: Tailwind CSS
- Animations: Framer Motion
- Smooth Scroll: Lenis

I feel like a portfolio, especially as a frontend developer — shouldn't just tell people you're good, it should make them feel it.
